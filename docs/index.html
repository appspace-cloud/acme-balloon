About

ACME Balloon is a Kubernetes deployment designed to manage cluster resources by creating low-priority pods that can be preempted when higher-priority workloads need resources.

  • Run with low priority (can be preempted)
  • Scheduled on spot/preemptible nodes
  • Help maintain cluster resource availability
  • Easily scalable based on cluster needs
